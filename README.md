# Shuangpin

在你的命令行中练习自然码双拼

![自然码双拼](assets/Natural_Code_Double_Pinyin_Scheme.png)

### 安装
```
go get github.com/A11Might/shuangpin
```

### 使用

键入 `shuangpin` 来开始练习自然码双拼，它会生成随机中文供你练习并显示你的 WPM 数据。
```
shuangpin
```
